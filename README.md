เป็นเกมที่ทดลองทำขึ้นมาเล่นๆ เก็บก้อนสี่เหลี่ยแล้ว ตัวจะยาวขึ้น



# ขั้นตอนการติดตั้ง 🔍

## 1. ติดตั้ง Pygame ก่อน
เปิด Command Prompt (CMD) แล้วพิมพ์:


	pip install pygame

1.1  **อัปเดต pip ก่อน **  
    พิมพ์คำสั่งนี้ในหน้าเดิมเลยครับ:

    python.exe -m pip install --upgrade pip

    
1.2  **ติดตั้ง setuptools เพื่อแก้ตัว Error ที่ฟ้องว่าหาไม่เจอ:**

    pip install setuptools
    
1.3  **ติดตั้ง pygame-ce แทนตัวเดิม:**

    pip install pygame-ce
